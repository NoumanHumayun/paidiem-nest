# Backend Paidiem

## This backend is available at https://paidiem-nest.herokuapp.com

in order to run this project locally clone the repo and run following scripts

## Installation

```bash
$ yarn
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Description

Nest Nased project with countries and search module
